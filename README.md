# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.


# installation

## Linux/MacOS
```
git clone git@github.com:LeoGoder/lazyvim_custom.git ~/.config/nvim
```

## Windows
```
git clone git@github.com:LeoGoder/lazyvim_custom.git $env:LOCALAPPDATA\nvim
```
